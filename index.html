Tankler

Tankler is a saltwater tank manager. It allows users to add livestock from the database to their tank, keep track of tank parameters, and create recurring tasks which will be displayed when due. It was built to help users take care of their saltwater fishtanks

Tankler is a fullstack app built using React/Redux for the front end and Node, Express, and Mongo/Mongoose for the back end
