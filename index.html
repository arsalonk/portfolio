  • Abjadify is a tool built to be used with the Quran. Each arabic letter has number values assigned to them, Abjadify uses both the heavenly and earthly abjad values to take the user inputted arabic text and return total sums of the text, sums for each word, and number values for every letter. Abjadify also informs the user whether the text they have inputted is divisible by either 9 or 7 and the mod 9 value of the text.
